Dailies
=======
A simple command line utility for keeping track of daily tasks.

Example Output
```
$ python dailies.py list
Task       | Total | Streak | Max Streak
Program    |     4 |      0 |     2
Paint      |     0 |      0 |     0
Read       |     2 |      1 |     1
Journal    |     2 |      9 |     9
```
